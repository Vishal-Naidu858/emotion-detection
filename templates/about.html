Technical Details

Model Architecture

  • 3 Convolutional Blocks (32→64→128 filters)
  • Batch Normalization after each conv layer
  • MaxPooling for spatial downsampling
  • Dropout for regularization (0.25–0.5)
  • Dense layers (256 → 128 → 7)
  • Softmax output for 7 classes

Dataset — FER-2013

  • 35,887 labeled face images
  • 48 × 48 pixels, grayscale
  • 7 emotion categories
  • Split: Train / Validation / Test
  • Data augmentation applied
  • Source: Kaggle

Backend Stack

  • Python 3.9
  • TensorFlow / Keras — model inference
  • OpenCV — face detection + preprocessing
  • Flask — web server and API routes
  • Haar Cascade — face localisation

Frontend Stack

  • Pure HTML5 — structure
  • CSS3 — styling and animations
  • Vanilla JavaScript — interactivity
  • Fetch API — async communication
  • No external JS frameworks needed
